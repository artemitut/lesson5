GIT flow

GIT clone - для того, чтоби "склонить" репозитор
GIT pull- для того, чтоби получить изменения с репозитором
GIT checkout -b branchname- создать новую ветку и переключиться на
GIT add .- добавить изменения, которие сделали локально
GIT commit -m "commit text"- подпись их
GIT push origin bramchname- запушить на git сервер
